# AGENTS.md

## Правила верстки

- Верстай аккуратно и визуально близко к референсу.
- Не используй изображение макета как единый background.
- Собирай интерфейс настоящими HTML/CSS-блоками.
- Страница должна быть адаптивной: desktop, tablet, mobile.
- Используй semantic HTML.
- Цвета, тени, обводки и размеры выноси в CSS variables.
- Для крупной типографики используй `clamp()`.
- Не добавляй лишний JS.
- После каждого этапа проверяй, что нет горизонтального скролла.
- Не переписывай весь проект без необходимости.
- Меняй только файлы, относящиеся к текущей задаче.
